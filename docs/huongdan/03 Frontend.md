# 03. Frontend

## Yêu cầu

- Node.js 18+
- npm 9+

## Chạy frontend

1. Vào thư mục frontend.
2. Cài dependencies và chạy:
   - `npm install`
   - `npm run dev`

Frontend chạy tại `http://localhost:5173`.
